Proje Konusu: E-Campus Activity Projesi

Kişiler:
- Administrator
- Öğrenci
- Akademisyen
- Kulüp Yönetici

Use-Cases:
- Etkinlik Listeleme -> Öğrenci
- Etkinliğe Kayıt Olma -> Öğrenci
- Etkinlik Oluşturma -> Akademisyen - Kulüp Yöneticisi
- Etkinlik Onaylama -> Akademisyen
- Kulüp Üyeleri Ekleme -> Kulüp Yöneticisi
- Bildirim Gönderme -> Administrator - Akademisyen 
- Raporlama -> Akademisyen
- Yönetim -> Administrator

Kullanılan Teknolojiler:
- Database: Redis, PostgreSQL(vb.)
- Backend: Java Spring Boot
- Frontend: Angular(Responsive)
- Mobil: Flutter
